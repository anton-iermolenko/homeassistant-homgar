# Old Release Files

This folder contains archived release notes and documentation from previous versions.

## Contents

- Release notes from v1.3.x series
- Old debugging documentation
- Historical publishing guides

## Purpose

These files are kept for reference but are not part of active development. They document the evolution of the integration and may be useful for:

- Understanding historical changes
- Debugging issues in older versions
- Reference for release process improvements

## Active Files

Current release files are kept in the root directory:
- `commit_message.txt` - Current release commit message
- `release_notes_vX.X.X.md` - Current release notes

These are temporary files that get cleaned up after release.
