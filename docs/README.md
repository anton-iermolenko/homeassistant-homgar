# Public Docs

This directory contains contributor-facing documentation for the HomGar / RainPoint Home Assistant integration.

Use these docs for stable project context:
- `architecture.md` — overall architecture, config flow, coordinator flow, entity creation, REST vs MQTT
- `testing.md` — test strategy, Docker validation, and payload fixture guidance
- `adding-device-support.md` — how to add support for new models safely
- `releasing.md` — version bump, changelog, validation, tag, and GitHub release flow
- `test-corpus.md` — inventory of real payload samples sourced from GitHub issues and device captures

Maintainer scratch notes, reverse-engineering artifacts, logs, and one-off scripts live in the local-only `internal_docs/` directory.
